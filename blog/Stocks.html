Introduction¶

In this project I wanted to look how to well we can predict the return on Goodyear (GT) stock based off of the return on other companies stock. The predictors I used were the return on the S&P 500 (SPY), Consumer Discretionary Select Sector (XLY), Ford (F), Toyota (T), and the exponentially weighted average of return on Ford and Toyota.

Data¶

I used Yahoo Finance as the source for data, since it makes historical stock data freely available. I chose 16 years worth of data and the common starting date of Aug. 1st 2000. Yahoo Finance provides the data to be downloaded in csv form. The data I used can be found in the "csv_files" directory.

Classes¶

stock_data class¶

The first class, stock_data, creates the pandas dataframe that will contain all the predictor stocks as well as the target stock values. The constructor will read in the raw data from the csv files and clean it. By cleaning, I mean it will lag the predictors/features so that they are a day behind the target. This elimates any ability of the model to "peak into the future." It also creates the predictors/features that are the exponentially weighted average of the stocks. The objects stores all these features and the target values into the pandas dataframe called "df."

The only other method in the class stock_data is "scale" function which scales the data and stores the mean and std of each feature column. It was important for me to encapsulate these two functions into a object, since I wanted to able to rescale the predicted results once the modeling was complete. By having the mean and std of features columns stored in the same object as the dataframe this made this process of rescaling the predicted values easier.

data_visualizer class¶

The data_visualizer class was just a container for different plotting mechanisms I would be using. While it is not necessary for such a small project, I wanted to maintain all the visualization methods into one object for future use in other projects.

class stock_data(object):
    
    # feature/predictor stock symbols
    features = ['SPY','EMA(F)','EMA(T)','XLY','F','T']
    
    # targer stock symbols
    target = 'GT'
    
    def __init__(self): 
        """
        This constructor will read and cleans the data 
        from the csv files.  Lagging and creating the exp.
        weight mean features.
        
        - creates self.df
        """
        # DATA VARIABLES
        self.df = None
        self.mean = None
        self.std = None
        

        # Read in the data from csv database
        data_F = pd.read_csv('csv_files/FORD.csv')
        data_T = pd.read_csv('csv_files/TOYOTA.csv')
        data_GT = pd.read_csv('csv_files/GOODYEAR.csv')
        data_SPY = pd.read_csv('csv_files/SPY.csv')
        data_XLY = pd.read_csv('csv_files/XLY.csv')

        # Lag the predictors compared to response so as to not
        # "peak into the future."

        # Drop the last day of predictors
        data_F.drop(data_F.index[0],inplace=True)
        data_T.drop(data_T.index[0],inplace=True)
        data_SPY.drop(data_SPY.index[0],inplace=True)
        data_XLY.drop(data_XLY.index[0],inplace=True)


        # Drop the first day of response
        last = data_GT.shape[0]-1
        data_GT.drop(data_GT.index[last],inplace=True)

        # Get the daily percent change i.e. daily return
        R_F = data_F['Adj Close'].pct_change()
        R_T = data_T['Adj Close'].pct_change()
        R_SPY = data_SPY['Adj Close'].pct_change()
        R_XLY = data_XLY['Adj Close'].pct_change()
        R_GT = data_GT['Adj Close'].pct_change()


        # Compute the exponentially weighted average of 
        # Ford and Toyoya over prev_days days
        prev_days = 30
        R_F_EMA = pd.Series.ewm(R_F,span=prev_days).mean()
        R_T_EMA = pd.Series.ewm(R_T,span=prev_days).mean()


        # Now drop the first 50 days to remove NANs from EMA data
        R_F.drop(R_F.index[0:prev_days],inplace=True)
        R_T.drop(R_T.index[0:prev_days],inplace=True)
        R_SPY.drop(R_SPY.index[0:prev_days],inplace=True)
        R_XLY.drop(R_XLY.index[0:prev_days],inplace=True)
        R_GT.drop(R_GT.index[0:prev_days],inplace=True)
        R_F_EMA.drop(R_F_EMA.index[0:prev_days],inplace=True)
        R_T_EMA.drop(R_T_EMA.index[0:prev_days],inplace=True)


        # Create the database, removing the first and last row
        # since they are NANs
        self.df = pd.DataFrame({'SPY':R_SPY,
                                'EMA(F)':R_F_EMA,
                                'EMA(T)':R_T_EMA,
                                'XLY':R_XLY,
                                'F':R_F,
                                'T':R_T,
                                'GT':R_GT})



        # first and last row have NANs
        self.df.drop(self.df.index[0], inplace=True)
        self.df.drop(self.df.index[self.df.shape[0]-1], inplace=True)

        # check to see if any NANs left
        #print df.isnull().values.any()
        

    def scale(self):
        """Scale the data so that the mean and std of the columns are
        0 and 1 respectively."""
        # standardize the data
        self.mean = self.df.mean()
        self.std = self.df.std()
        
        self.df = (self.df - self.mean/self.std)

class data_visualizer(object):
    ''' This class is just a container for different visualizations
    of the data stored in stock_dat object.'''
    
    def __init__(self):
        """Empty constructor."""
        
    def corr(self, data):
        """Makes a heatmap of the correlation matrix."""
        sns.heatmap(data.df.corr())
    
    def simple_least_squares(self, data):
        """Plots the simple least squares regression lines
        with confidence intervals for each of the features."""
        sns.pairplot(data.df, x_vars=data.features,
                     y_vars=data.target, kind='reg', size=8)

Conclusion¶

Overall linear modeling using multiple linear regression is not a great model for predicing returns on Goodyear tires from knowledge of the selected predictor companies. Linear regression without regularization and with $L_1$ regularization (Lasso) perform about equally well. Overall they able to explain about 35% of the variance in the training set data and percentage of variance explained on the test set data is expected to be even smaller. More complicated models should be considered and adding in addational features is also necessarry to improve the predictive power.
