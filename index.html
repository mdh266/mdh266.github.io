About Me!

I recently defended my Ph.D in Computational Applied Mathematics and am currently looking for employment as a Data Scientist in New York City.

As a graduate student at the University of Texas at Austin my research focused on the intersection of math, programming and engineering. I developed high performance software in C++ for chemists to numerically model solar cells and optimize the cell design. This involved creating a mathematical model of the way solar cells work and then writing code to make predictions from this model on efficiency of different designs. The bulk of this code and information on the project can be found on at: PECS

As a data science fellow at Insight Data Science I built a Python based web application for exploring crime rates in New York City. This project involved extracting, cleaning and manipulating nine years of geospatial data as well as looking into appropriate predictive models such as regression, random forests and time series methods. The web application can be found on the website: CrimeTime.

deal.ii Finite Element Library

Contributed to the open source deal.ii C++ library by adding functionality to its interface with the Trilinos library. Specifically, I added the ability to use direct solvers for large-scale distributed linear systems (using MPI) without the need to re-factorize the matrix every time a solve is performed. This improved runtimes for codes where one is solving linear systems: Ax=b, where A is fixed, but there are multiple b's and in my case resulted in a 250x speed up. I also added unit test to test-suite for testing this functionality.

Semiconductor Device Simulator

This code is designed to numerically solve the drift-diffusion-Poisson equations for semiconductor devices in 2D and 3D using the deal.ii C++library. The code runs in parallel using multithreading through the Intel Thread Building Blocks. Poisson's equation is solved using the mixed finite element method. The drift-diffusion equations for electron and hole transport are discrititzed with the local discontinuous Galerkin method and time stepping is performed using implicit-explicit (IMEX) methods.
